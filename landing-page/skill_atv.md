Crie uma Landing Page em React (JavaScript) inspirada visualmente no site https://www.grefeletrica.com.br/.

O objetivo é desenvolver uma landing page institucional moderna, organizada e responsiva para uma empresa de serviços elétricos.

A aplicação deve utilizar componentes React separados para cada seção da página.

Estrutura obrigatória:

* Header
* Hero Section
* Sobre a Empresa
* Serviços
* Depoimentos
* Preços (Pricing)
* Contato
* Footer

Requisitos:

### Header

* Logo
* Menu de navegação
* Botão de orçamento

### Hero

* Título principal
* Texto de apresentação
* Botão de ação
* Imagem ilustrativa

### Sobre

* Breve descrição da empresa
* Imagem de apoio

### Serviços

Exibir 3 a 6 serviços em cards contendo:

* Ícone
* Título
* Descrição

### Depoimentos

Exibir pelo menos 3 depoimentos de clientes.

### Pricing

Criar 3 planos ou categorias de serviço:

* Residencial
* Comercial
* Industrial

Cada card deve possuir:

* Nome
* Descrição
* Lista de benefícios
* Botão de contato

### Contato

Criar formulário contendo:

* Nome
* E-mail
* Telefone
* Mensagem

Preparar a estrutura para integração com Netlify Functions para envio de e-mail.

### Footer

* Logo
* Links rápidos
* Informações de contato
* Copyright

### Responsividade

O site deve funcionar corretamente em:

* Desktop
* Tablet
* Mobile

### Organização

Utilizar a seguinte estrutura:

src/
components/

* Header
* Hero
* About
* Services
* Testimonials
* Pricing
* Contact
* Footer

pages/

* Home

Criar CSS separado para cada componente.

Priorizar código simples, organizado, reutilizável e semelhante ao que normalmente seria desenvolvido em sala de aula em uma disciplina de Desenvolvimento Web utilizando React.
